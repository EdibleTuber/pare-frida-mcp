  ## Summary

  v1 Android slice of `pare-frida-mcp` — a Python/FastMCP stdio worker that
  gives the PARE agent Frida 17 dynamic-instrumentation tools. 13 tools
  (devices, attach, java_hook/java_hook_remove, memory r/w/scan/enum,
  execute_script, search/read_capture); SQLite per-session capture store
  with promoted indexed columns + FTS5 and blob spill; bundled JS agent.

  ## Gating (read this if nothing else)

  Worker declares per-tool `risk_tier` in `list_tools` (passes
  `agent_core`'s `assert_conformance`). PARE's runtime `RiskAwareToolPool`
  currently sources the declared tier from worker-wide `risk_default` +
  name-pattern overrides; **the `risk_overrides:` workers.yaml parser is
  a PARE-side follow-up not yet shipped**. Until it ships, this worker is
  gated only at `risk_default: low`. Treat `execute_script` and
  `write_memory` accordingly.
